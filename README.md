## 📦 Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/tony-electronics.git
   cd tony-electronics
   ```

2. **Database Setup**
   - Import `shop_db.sql` into your MySQL database
   - Update database credentials in `components/connect.php`

3. **Server Setup**
   - Use XAMPP, WAMP, or any PHP server
   - Place project in your web root directory

4. **Configure Connection**
   - Edit `components/connect.php` with your database details

5. **Access the Application**
   - User Portal: `http://localhost/tony-electronics/home.php`
   - Admin Portal: `http://localhost/tony-electronics/admin/admin_login.php`
